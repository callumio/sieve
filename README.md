# sieve

> The web, strained.

My personal uBlock Origin filter list, auto-built daily from custom rules and
third-party sources.

## Subscribe

```text
https://raw.githubusercontent.com/callumio/sieve/main/dist/filter.txt
```

Paste into uBlock Origin -> Dashboard -> Filter lists -> Import.

## Third-party licenses

Each block in `dist/filter.txt` is annotated with its source URL and license.
See `lists.txt` for the full list.
